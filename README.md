# zdbc

A performance oriented [ODBC](https://github.com/microsoft/ODBC-Specification) & [Apache Arrow](https://arrow.apache.org) library for Zig

## Development

```shell
> nix develop -c $SHELL
```

```shell
> make
```

```shell
> make test
```

```shell
> make build
> make build.fast
> make build.small
> make build.safe
```

```shell
> make run
```
